# Healthcare Cost & Risk Prediction using Machine Learning

## 📌 Project Overview
This project applies machine learning techniques to analyze healthcare data and predict medical costs as well as classify high-risk patients.

## 🎯 Objectives
- Predict medical insurance charges (Regression task)
- Classify patients into high-cost / low-cost groups (Classification task)

## 📊 Dataset
Medical Cost Personal Dataset (Insurance dataset)
Features include:
- age
- sex
- bmi
- children
- smoker
- region
- charges

## 🧠 Methods Used
- Exploratory Data Analysis (EDA)
- Data preprocessing (One-hot encoding)
- Linear Regression model
- Logistic Regression model

## 📈 Results
- Regression R² Score: ~0.78
- Classification Accuracy: ~0.94

## 💡 Key Insights
- Smoking is the strongest factor affecting medical costs
- Age and BMI significantly influence insurance charges
- Machine learning can effectively predict high-cost patients

## 🛠 Tools & Libraries
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
